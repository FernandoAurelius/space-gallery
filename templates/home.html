Proposta do projeto

Eu desenvolvi o “Space Gallery” com o objetivo de criar um ponto de encontro digital para todos que se encantam com o céu estrelado. Escolhi o Django por sua robustez e eficiência, o que me permitiu construir uma aplicação web que é ao mesmo tempo dinâmica e fácil de usar, seguindo uma lógica clara com o modelo MVT.

Na construção da aplicação, utilizei HTML para definir sua estrutura e CSS para dar um estilo visual limpo e moderno. O Bootstrap foi essencial para criar uma interface que responde bem em diferentes dispositivos. Para o controle de versão, optei pelo Git, que me ajuda a manter um histórico organizado das mudanças, e disponibilizei o código no GitHub, facilitando a colaboração e o compartilhamento do projeto.

Tecnologias Utilizadas

Este projeto foi desenvolvido com uma variedade de tecnologias modernas para garantir a melhor experiência possível:

  • Django
    Framework de desenvolvimento web rápido e eficiente do Python, usado para definir toda a lógica da aplicação, segundo o Modelo MVT
  • HTML
    Linguagem de Marcação de Hipertexto, usada pra definir a estrutura visível da aplicação
  • CSS
    Linguagem de Estilização em Cascata, usada pra definir o estilo da interface da aplicação
  • Bootstrap
    Framework front-end que fornece estruturas CSS para desenvolvimento rápido de aplicações web
  • Git
    Software de controle de versionamento de código, usado para rastrear e versionar o desenvolvimento do projeto, desde seu início.
  • GitHub
    Plataforma de compartilhamento de repositórios Git, usada para hospedar o código-fonte da aplicação.
  • Fly.io
    Platform-as-service que oferece facilidade no momento do deploy para diversas linguagens e modelos de aplicação, incluindo o Django

Funcionalidades do Projeto

A proposta do projeto "Space Gallery" é a de trazer imagens de qualidade para os amantes do céu noturno, de seus mistérios e segredos. Possui funcionalidades excepcionais e ainda mais, incluindo:

🚀 Upload de Fotografias Espaciais
💫 Visualização de uma galeria completa e compartilhada do Universo
🗑️ Deleção de fotos
🖌️ Edição de fotos
📋 Inclusão de descrições, legendas e fontes detalhadas das fotos

👤 Login
💨 Logout
